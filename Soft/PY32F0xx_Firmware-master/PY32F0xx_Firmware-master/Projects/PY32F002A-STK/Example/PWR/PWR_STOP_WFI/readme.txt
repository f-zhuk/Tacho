================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了stop模式下，GPIO外部中断唤醒功能

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU；
2. 断开swd连接线并重新上电;
3. LED灯亮，按下按键，LED灯灭，串口打印"STOP MODE!",进入stop模式；
4. 在PA06上产生一个下降沿，程序退出stop模式，并且串口打印"WAKEUP OK!"；
5. LED以200ms的间隔进行翻转。

================================================================================
注意事项：
STK板        USB转TTL模块
PA2(TX) --> RX
PA3(RX) --> TX
GND      --> GND
波特率：115200bps

演示此样例功能时需要断开swd连接线，因为默认情况下，仿真器会把DBGMCU_CR.DBG_STOP置位
注意关闭systick中断，防止该中断唤醒MCU

================================================================================
