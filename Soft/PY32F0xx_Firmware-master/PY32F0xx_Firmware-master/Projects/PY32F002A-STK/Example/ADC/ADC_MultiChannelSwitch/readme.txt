================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了ADC的多通道切换。

================================================================================
测试环境：
测试用板：PY32F002_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译并下载程序；
2. 串口先打印一次通道5(PA5)的转换值
3. 之后切换到通道4(PA4)转换并打印其转换值.
4. 1s后又切换回通道5

================================================================================
注意事项：
STK板        USB转TTL模块
PA02(TX) --> RX
PA03(RX) --> TX
GND      --> GND
波特率：115200bps
================================================================================
