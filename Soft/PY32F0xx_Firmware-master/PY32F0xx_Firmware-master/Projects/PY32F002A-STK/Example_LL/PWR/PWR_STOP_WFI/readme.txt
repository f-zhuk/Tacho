================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了通过 WFI(wait for interrupt)指令进入stop模式，使用GPIO中断唤醒

================================================================================
测试环境：
测试用板：PY32F002A_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU；
2. 断开swd连接线并重新上电;
3. LED灯亮，按下按键，LED灯灭，进入stop模式；
4. 在PA06上产生一个下降沿，程序退出stop模式；
5. LED以200ms的间隔进行翻转。
================================================================================
注意事项：
1，演示此样例功能时需要断开swd连接线并重新上电，因为默认情况下，仿真器会把
DBGMCU_CR.DBG_STOP置位。

================================================================================
