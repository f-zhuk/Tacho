================================================================================
                                样例使用说明
================================================================================
功能描述：
此样例演示了使用TIM1输出频率为10Hz占空比为50%的PWM波形及其互补信号，并实现了250ns
的死区和刹车功能。

================================================================================
测试环境：
测试用板：PY32F003_STK
MDK版本： 5.28
IAR版本： 9.20
================================================================================
使用步骤：
1. 编译下载程序到MCU，并运行
2. 将PA6引脚接低电平
3. 使用逻辑分析仪观察引脚PA3，PA7的波形
4. 将PA6引脚接高电平，PWM停止输出

================================================================================
注意事项：
CH1  -> PA3 占空比：50%
CH1N -> PA7 占空比：50%

================================================================================
